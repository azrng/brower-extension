# Boss 职位技能分析插件 - 使用说明

## 项目概述

这是一个基于 Blazor WebAssembly 开发的 Edge 浏览器插件，用于在 Boss 直聘搜索页面自动分析职位技能要求。

### 核心功能

- ✅ 在 Boss 直聘搜索页注入右侧悬浮侧栏
- ✅ 自动翻页收集职位信息（前 N 页）
- ✅ 逐个打开职位详情页提取内容
- ✅ 使用兼容 OpenAI 规范的 AI 接口进行技能分析
- ✅ 两阶段分析：单职位分析 + 全局汇总
- ✅ 支持技能分类、频次统计、同义词归一
- ✅ 导出分析结果为 JSON 文件

## 技术栈

- **前端框架**: Blazor WebAssembly
- **插件模板**: Blazor.BrowserExtension v4.0.2
- **目标浏览器**: Microsoft Edge (Manifest V3)
- **目标站点**: Boss 直聘 (www.zhipin.com)
- **AI 接口**: 兼容 OpenAI Chat Completions API

## 开发指南

### 环境要求

- .NET 10.0 SDK
- Microsoft Edge 浏览器
- Visual Studio 2022 或 VS Code（带 C# 扩展）

### 安装开发模板

```bash
dotnet new install Blazor.BrowserExtension.Template
```

### 构建项目

```bash
cd src
dotnet build
```

构建输出位于 `bin/Debug/net10.0/browserextension/` 目录。

### 加载到浏览器

1. 打开 Edge 浏览器
2. 访问 `edge://extensions/`
3. 启用"开发人员模式"
4. 点击"加载已解压的扩展"
5. 选择项目的 `bin/Debug/net10.0/browserextension/` 目录

### 调试

- **Blazor 组件**: 使用浏览器开发者工具的 Console 查看日志
- **Content Script**: 在 Boss 直聘页面的 Console 中查看
- **Background Worker**: 在扩展管理页面的"服务工作进程"中查看

## 使用说明

### 1. 配置 AI 接口

在右侧侧栏中填写以下配置：

- **API Base URL**: OpenAI 兼容接口地址，如 `https://api.openai.com/v1`
- **API Key**: 你的 API 密钥
- **Model**: 模型名称，如 `gpt-3.5-turbo` 或 `gpt-4`
- **前几页**: 最多处理前几页搜索结果（默认 3 页）
- **超时(秒)**: API 请求超时时间（默认 30 秒）
- **处理间隔(毫秒)**: 职位之间的处理间隔（默认 2000 毫秒）

点击"保存配置"按钮保存设置。

### 2. 开始分析

1. 访问 [Boss 直聘搜索页面](https://www.zhipin.com/)
2. 搜索你感兴趣的职位关键词
3. 点击右侧侧栏的"开始识别"按钮
4. 插件将自动：
   - 读取当前页职位列表
   - 翻页收集更多职位（根据配置的前几页）
   - 逐个打开职位详情页
   - 提取经验要求和岗位描述
   - 调用 AI 进行技能分析
   - 汇总所有结果

### 3. 查看结果

分析完成后，侧栏将显示：

- **技能关键词**: 提取的所有技能标签
- **技能分类**: AI 自动分类的技能组
- **技能频次**: 每个技能出现的次数
- **同义词归一**: 原词到标准词的映射
- **总体总结**: 岗位技能画像的文字描述

### 4. 导出 JSON

点击"导出 JSON"按钮，将分析结果下载为 JSON 文件。

导出的 JSON 结构：

```json
{
  "keywords": ["Java", "Spring", "MySQL", ...],
  "categories": [
    {
      "name": "后端开发",
      "skills": ["Java", "Spring", "MyBatis"]
    }
  ],
  "frequencies": {
    "Java": 15,
    "Spring": 12,
    "MySQL": 10
  },
  "synonymMappings": {
    "JS": "JavaScript",
    "react": "React"
  },
  "summary": "该职位主要要求 Java 后端开发经验...",
  "meta": {
    "processedPages": 3,
    "totalJobs": 30,
   modelName": "gpt-3.5-turbo",
    "timestamp": "2024-03-20T12:00:00"
  }
}
```

## 注意事项

### 页面选择器

插件中的 Boss 直聘页面选择器可能需要根据实际页面结构调整：

**ContentScript.js** 中的选择器：

```javascript
// 职位列表
document.querySelectorAll('.job-card-wrapper')

// 职位详情
document.querySelector('.job-sec-text')
```

如果 Boss 直聘更新页面结构，可能需要调整这些选择器。

### 反爬虫限制

- 插件默认设置了 2000 毫秒的处理间隔，避免触发反爬虫
- 如果遇到问题，可以适当增加处理间隔
- 建议使用代理或降低并发数

### AI 配额

- 每个职位会调用一次 AI 接口
- 最后还会调用一次 AI 进行汇总
- 请确保你的 API 配额充足

### 隐私和安全

- 插件只在本地浏览器中运行，不上传数据到第三方服务器
- 职位信息仅通过配置的 AI 接口处理
- API Key 存储在浏览器本地存储中

## 故障排除

### 插件无法加载

1. 检查是否使用 Edge 浏览器
2. 确认已启用"开发人员模式"
3. 查看浏览器控制台的错误信息

### 无法提取职位信息

1. 确认在 Boss 直聘搜索页面
2. 检查页面是否完全加载
3. 查看浏览器控制台是否有错误

### AI 调用失败

1. 检查 API Base URL 是否正确
2. 确认 API Key 有效
3. 检查网络连接
4. 查看浏览器控制台的详细错误信息

### 侧栏不显示

1. 刷新页面
2. 检查是否有 JavaScript 错误
3. 重新加载插件
